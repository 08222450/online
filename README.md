# online

软件工程小组实验，在线考试管理系统

files文件夹存储头像等图片，springboot存储后端内容，vue存储前端内容（前后端分离，建议两个文件夹分别用idea打开）。

前端打包地址位于：`/vue/dist`，后端打包地址位于：`/springboot/target`。
